# Faith Cosy Homes

A static landing page for Faith Cosy Homes, a luxury Airbnb-style accommodation in Nyali, Mombasa.

## Overview

- **Live site:** static HTML/CSS/JavaScript
- **Location:** Nyali, Mombasa, Kenya
- **Purpose:** showcase premium short-term stay amenities and direct booking via WhatsApp

## Files

- `Index.html` — home page content and layout
- `Style.css` — styling for hero, sections, navigation, and responsive design
- `Script.js` — mobile menu toggle and small interactive behavior
- `vercel.json` — Vercel deployment configuration

## Deployment

This project is ready for deployment on Vercel as a static site. The root contains the production `index.html`, so no build step is required.

To deploy:

1. Connect your repository to Vercel
2. Set the root directory to the project root
3. Deploy the site

## Notes

- `.gitignore` excludes local editor files and temporary build artifacts
- Keep `vercel.json` at the project root for clean Vercel detection
